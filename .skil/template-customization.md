# Template Customization Skill

## Overview

This skill provides guidelines for customizing MOVTIGROUP templates for different projects while maintaining design consistency.

## Guidelines

### 1. Maintain Design Language
- Keep the modern, clean aesthetic
- Use the established color palette
- Follow typography guidelines from DESIGN.md

### 2. Preserve Structure
- Maintain semantic HTML5 structure
- Keep BEM naming conventions
- Preserve responsive breakpoints

### 3. Customization Areas
- **Colors:** Modify CSS custom properties (variables)
- **Content:** Replace placeholder text and images
- **Fonts:** Update font families while maintaining readability
- **Layout:** Adjust container widths if needed

### 4. Do Not Modify
- Core CSS architecture
- JavaScript functionality
- Accessibility features
- RTL/LTR support logic

## Process

1. Review DESIGN.md for design system specifications
2. Make changes in a separate branch
3. Test across all breakpoints
4. Verify accessibility
5. Update documentation
