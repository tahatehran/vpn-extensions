# MOVTIGROUP Skills

This directory contains skill definitions for AI assistants and automation tools working with the MOVTIGROUP project.

## Available Skills

### 1. Template Customization
**File:** `template-customization.md`

Guidelines for customizing MOVTIGROUP templates for different projects while maintaining design consistency.

### 2. Responsive Design
**File:** `responsive-design.md`

Best practices for ensuring responsive behavior across all devices and screen sizes.

### 3. Bilingual Implementation
**File:** `bilingual-implementation.md`

Guidelines for implementing RTL (Persian) and LTR (English) layouts correctly.

### 4. Component Development
**File:** `component-development.md`

Standards for creating new reusable components within the design system.

### 5. Accessibility Audit
**File:** `accessibility-audit.md`

Checklist and procedures for ensuring WCAG 2.1 compliance.

## Usage

Skills can be referenced by AI assistants to understand project-specific requirements and constraints.
